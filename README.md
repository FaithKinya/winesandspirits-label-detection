A computer vision project that detects alcohol bottle labels in images using YOLOv8 and reads the text on the labels using OCR. The system can identify the brand of the bottle from an image, providing a seamless pipeline from detection to text extraction.

## Features

- Detects bottle labels from images using YOLOv8.

- Extracts text from detected labels using EasyOCR.

- Returns the brand name from the text using a simple text matching method.

- Supports batch processing of multiple bottle images.

- Outputs cropped bottle images and detected text for easy verification.
